Pinchzoom.js

Lets you zoom DOM elements using the multi-touch gestures known from iOS devices.
Like pinchzoom, doubletab zoom, draging.

  • Source
  • Example

Device support

It was developed for a webbased iPad application. So its well tested on mobile Safari for iOS devices. Other devices might work as well but are currently not tested.

Pinchzoom.js depends on Underscore.js and jQuery.

Swipe

Pinchzoom.js does careful event bubbling. Only self used touch events are prevented from bubbling up in the DOM. This makes it possible to combine pinchzoom with other libraries like swipe.js.
See demo

Hardware acceleration

Uses 3d CSS transformation for fast animations.

AMD

AMD is supported. You can use a module loader like require.js. But script tag include is supported as well.
